# [Knightly Survival](https://douglasrbto.itch.io/knightly-survival) ![](https://badgen.net/static/Status/Complete/green)

## Objectives
- Explore
- Kill enemies to earn XP
- Collect meat to recover HP
- Survive!

## About

Small game made for a bootcamp project.
